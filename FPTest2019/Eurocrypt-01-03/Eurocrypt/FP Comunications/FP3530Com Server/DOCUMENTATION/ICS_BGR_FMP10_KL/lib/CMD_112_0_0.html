	Област за данни:	
			OperatorNumber
	Отговор:	
			Receipts
			Total
			Discount
			Surcharge
			Void
			Name

	RegisteredSalesNumber	Брой продажби.
	TotalAccumulatedSum	Обща натрупана сума.
	DiscountsNumber		Брой отстъпки.

	OperatorName		Име на оператора.
