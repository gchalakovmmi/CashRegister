65h (101)	ЗАДАВАНЕ НА ОПЕРАТОРСКА ПАРОЛА

	Област за данни:	
			OpCode
			OldPwd
			NewPwd
	Отговор:		
			Няма данни

	OpCode		Код на оператор. От 1 до 30.
	OldPwd		Стара парола (1 до 8 цифри).
	NewPwd		Нова парола (1 до 8 цифри).

Задава една от тридесетте операторски пароли. 
Паролата ще бъде поискана при отваряне на фискален бон. 
След инициализация или нулиране на оперативната памет паролите са същите като номера на оператора.

Parameter Parameter mode Parameter type Description
OpCode [IN] [STRING] 
OpCode		Код на оператор. От 1 до 30.
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing input parameter.
-10511 Проверка дали входния параметър превишава дадена горна граница със стойност от тип цяло число. Wrong input data.
-10512 Проверка дали входния параметър превишава дадена долна граница със стойност от тип цяло число. Wrong input data.

OldPwd		Стара парола (1 до 8 цифри).

-10502 Проверка за превишаване на горна граница на дължината във входния параметър. Wrong length - input data.
-10503 Проверка за превишаване на долна граница на дължината на входния параметър. Wrong length - input data.

NewPwd		Нова парола (1 до 8 цифри).
