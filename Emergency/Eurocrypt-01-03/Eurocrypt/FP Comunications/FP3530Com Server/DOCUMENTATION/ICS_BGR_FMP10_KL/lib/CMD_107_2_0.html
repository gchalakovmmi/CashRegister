CMD_107_2_0

 

6Bh (107)	ДЕФИНИРАНЕ И ЧЕТЕНЕ НА АРТИКУЛИ


	Входни данни:	
			PLU
			Quantity
	Отговор:	
			ErrorCode

	PLU		Номер на артикула (1 до 999999999).
	Quantity		Корекция на количеството - плаващо число с до 3 десетични знаци. 
			Положителните числа увеличават количеството, отрицателните го намаляват.


	ErrorCode	Един байт, показващ резултата от операцията, със значение:
			•	'P	Командата е успешна.
			•	'F'	Командата е неуспешна.


Забележка:

Промяна на наличното количество за артикул.
Командата се отхвърля ако артикулът не е програмиран.

Parameter Parameter mode Parameter type Description
PLU [IN] [STRING] 
PLU		Article number (1 to 999999999).
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing input parameter.
-10511 Проверка дали входния параметър превишава дадена горна граница със стойност от тип цяло число. Wrong input data.
-10512 Проверка дали входния параметър превишава дадена долна граница със стойност от тип цяло число. Wrong input data.
Quantity [IN] [STRING] 
Quantity		Quantity correction - a floating-point number with 3 decimal places. Positive number increases the available 
		quantity, negative decreases it.
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing input parameter.
-10505 Проверка за това дали входния параметър е реално число. Wrong input data.
ErrorCode [IN/OUT] [STRING] 
ErrorCode

Един байт, показващ резултата от операцията, със значение:
	•	'P	Командата е успешна.
	•	'F'	Командата е неуспешна.
Error value Verification type Description
-10524 Проверка за наличие на изходящ параметър. Wrong output data.
-10515 Проверка дали изходния параметър е със стойност от даден изброен списък от стойности. Wrong output data.
