CMD_50_0_0

 

32h (50)	ДАНЪЧНИ СТАВКИ ЗАДАВАНИ ПРЕЗ СЪОТВЕТНИЯ ПЕРИОД

Командата разпечатва отчет на промените на десетичните знаци и данъчните ставки за посочения период.

	Област за данни:
		StartDate
		EndDate
	Отговор:
		Няма

	StartDate		Дата за начало на периода - DDMMYY /6 bytes/.
	EndDate		Дата за край на периода - DDMMYY /6 bytes/.

Parameter Parameter mode Parameter type Description
StartDate [IN] [STRING] 
StartDate		The starting date of the period - DDMMYY/6 bytes/
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing parameter.
-10502 Проверка за превишаване на горна граница на дължината във входния параметър. Wrong data length.
EndDate [IN] [STRING] 
EndDate The end date of the period - DDMMYY /6 bytes/
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing parameter.
-10502 Проверка за превишаване на горна граница на дължината във входния параметър. Wrong data length.
