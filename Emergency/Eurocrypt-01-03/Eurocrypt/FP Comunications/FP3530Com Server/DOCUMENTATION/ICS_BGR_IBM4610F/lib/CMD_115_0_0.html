73h (115)	ПРОГРАМИРАНЕ НА ГРАФИЧНО ЛОГО

	Област за данни:	
			RowNum
			RowData

	Отговор:
			Няма

	RowNum		Показва реда, който програмираме. Число от 0 до 199.
	RowData		Графични данни. 
			Задават се в шестнайсетичен вид, два символа за всеки байт информация. 
	

Командата дава възможност за дефиниране на графично лого с размер до 72x25mm (576x200 точки) по  желание на потребителя. 
Редът на програмиране трябва да е следният:
Изпраща се команда за изтриване ('E').
Изпращат се графичните данни от ред 0 до последния ред от графиката последователно. 
За ширина на графичното лого принтерът взема най-широкия ред.
Изпраща се команда 'P' с височината на графиката. При това данните се прехвърлят от оперативната памет във флаш-паметта на 
печатащото устройство.

Печатът на логото може да се разрешава или забранява с команда 43. 
Логото не се губи при изключване на принтера или нулиране на RAM.

Parameter Parameter mode Parameter type Description
RowNum [IN] [STRING] 
RowNum		Shows the line, which is being programmed - a number between 0 до 199
