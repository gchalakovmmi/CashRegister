CMD_107_12_1

 

6Bh (107)	ДЕФИНИРАНЕ И ЧЕТЕНЕ НА АРТИКУЛИ


Намиране на последния свободен (непрограмиран) артикул с номер по-малък или равен на PLU.
ANSWER_PLU е първия свободен (непрограмиран) артикул с номер по-малък или равен на PLU. 

	Входни данни:	
			PLU
	Отговор:	
			ErrorCode
			ANSWER_PLU


	ErrorCode	Един байт, показващ резултата от операцията, със значение:
			•	'P	Командата е успешна.
			•	'F'	Командата е неуспешна.
	
	ANSWER_PLU	Номер на артикула. Между 1 и 999999999.

Parameter Parameter mode Parameter type Description
PLU [IN] [STRING] 
PLU	Номер на артикула. Между 1 и 999999999.
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing input parameter.
-10510 Проверка за това дали входния параметър е цяло число. Wrong input data.
-10511 Проверка дали входния параметър превишава дадена горна граница със стойност от тип цяло число. Wrong input data.
-10512 Проверка дали входния параметър превишава дадена долна граница със стойност от тип цяло число. Wrong input data.
ErrorCode [IN/OUT] [STRING] 
ErrorCode

Един байт, показващ резултата от операцията, със значение:
	•	'P	Командата е успешна.
	•	'F'	Командата е неуспешна.
Error value Verification type Description
-10524 Проверка за наличие на изходящ параметър. Missing output parameter.
-10515 Проверка дали изходния параметър е със стойност от даден изброен списък от стойности. Wrong output data.
ANSWER_PLU [IN/OUT] [STRING] 
ANSWER_PLU	Номер на артикула. Между 1 и 999999999.
Error value Verification type Description
-10524 Проверка за наличие на изходящ параметър. Missing output parameter.
-10521 Проверка за това дали изходящия параметър е цяло число. Wrong output data.
-10522 Проверка дали изходния параметър превишава дадена горна граница със стойност от тип цяло число. Wrong output data.
-10523 Проверка дали изходния параметър превишава дадена долна граница със стойност от тип цяло число. Wrong output data.
