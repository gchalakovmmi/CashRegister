49h (73) 	ОТПЕЧАТВАНЕ НА ФИСКАЛНАТА ПАМЕТ ПО НОМЕР НА БЛОК

	Област за данни:	
		StartRec
		EndRec
	Отговор:	
		Няма


	StartRec	Номер на началния фискален блок. 4 байта.
	EndRec	Номер на крайния фискален блок. 4 байта.

Тази команда води до отпечатване на пълен отчет на фискалната памет от даден номер до друг. 
Ако няма входни данни, приема се целия перод на експлоатация.

Parameter Parameter mode Parameter type Description
StartRec [IN] [STRING] 
StartRec	The number of the starting fiscal entry - 4 bytes
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing input parameter.
-10510 Проверка за това дали входния параметър е цяло число. Wrong input data.
EndRec [IN] [STRING] 
EndRec	The number of the ending fiscal entry - 4 bytes
Error value Verification type Description
-10500 Проверка за наличие на входящ параметър. Missing input parameter.
-10510 Проверка за това дали входния параметър е цяло число. Wrong length - input data.
